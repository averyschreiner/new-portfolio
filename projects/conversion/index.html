Unit Conversions
Description
This project is the web implementation of the popular C# library, UnitsNet. UnitsNet offers conversions for just about any measurement you could possilby need. This was my first ASP.NET Core project, so this was mostly me grasping the MVC ideas as well as C# syntax.
Main Takeaways
  • C# Introduction
  • ASP.NET Core
  • Solidified understanding of MVC design pattern
  • jQuery for updating options based on main selection

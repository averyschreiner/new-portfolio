Virtual Assistant gpt-assistant.net
Description
My virtual assistant is a website that uses OpenAI's GPT 3.5 to carry conversations on just about any topic. This site allows for easy interaction with GPT 3.5 by providing speech to text input and text to speech output in 14 languages. There are also lots of customizable options such as assistant name, assistant personality, whether the assistant should use sir, ma'am, or none when addressing with you, and adjustable colors for the webpage. I added some helpful shortcuts, such as current weather commands, article summaries given an article link, and opening spotify with a requested song. Lastly, this website has a sign-in option that will save a users previous conversations and settings.
Main Takeaways
  • Using Flask.
  • Passing and receiving JSON.
  • Google authentication for sign-in.
  • Google App Engine deployment.
  • Google Text-to-Speech and Speech-to-Text.
  • Firebase for users' previous conversations and settings.
  • The importance of virutal environments.
